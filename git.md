# git
```
git init    初始化git
git config use.email ""   ""git的账号
git config use.name ""    ""git的用户名
git commit -a -m "init"  ""内容为提交内容的备注
git config --global core.autocrlf false
git status     获取当前代码状态
git clone url    从远程clone代码到本地，地址为url
git checkout 恢复已修改的文件
git rm --cached 指定删除的文件
向远程仓库推送
git remote add origin https://github.com/123456kylin/first.git
git push -u origin master

```
