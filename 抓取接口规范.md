2.书籍信息
```
存储进数据库
参数名称                描述信息
articleid              书籍id
articlename            书籍名称
author                 作者
intro                  内容简介
keywords               关键字
size                   全书字数
fullflag               书状态（是否完结）
sortid                 分类id
chapters               当前章节数
cover                  封面地址
postdate               发表时间
lastupdate             更新时间

数据格式
[
{
    "articleid" => ,
    "articlename" => '123',
    "author" =>,
    "intro" => ,
    "keywords" => ,
    "size" => ,
    "fullflag" => ,
    "sortid" => ,
    "chapters" => ,
    "cover" => ,
    "postdate" => ,
    "lastupdate" => 
}
]
        
```
